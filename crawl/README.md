### Di chuyển vào folder crawl

```
cd crawl
```

### Tạo môi trường:

```bash
python3 -m venv venv
```

### Kích hoạt môi trường:

```bash
source venv/bin/activate
```

### Cài đặt các thư viện cần thiết:

```bash
pip install -r requirements.txt
```

### Chạy file python

```
python3 vnu_wikipedia.py
```
